# Start Here

Before diving into the stages, make sure you're set up for success.

## Prerequisites

- **Programming**: comfortable with Python (functions, classes, virtual environments).
- **Command line**: basic terminal navigation and running scripts.
- **Git**: cloning, committing, and pushing to GitHub.
- **Math (light)**: intuition for vectors, probability, and gradients helps but isn't required to start.

## Recommended setup

```bash
# Create an isolated environment for your experiments
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
```
