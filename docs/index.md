# AI Engineering Roadmap

Welcome to my public, hands-on roadmap for becoming an **AI Engineer** — a living
knowledge base that grows as I learn and build. Each stage collects notes,
worked examples, and references on a core theme, from first principles up to
production systems.

!!! tip "How to use this site"
    Start with **[Start Here](start-here/index.md)** to check the prerequisites,
    then work through the stages in order. Use the search bar (top right) or
    browse by **tag** to jump straight to a concept like RAG, agents, or LLMOps.

## The stages

| Stage | Theme | What you'll learn |
| ----- | ----- | ----------------- |
| [Stage 0](stage-0-foundations/index.md) | Foundations | Python, ML basics, how LLMs work |
| [Stage 1](stage-1-rag/index.md) | RAG | Retrieval-augmented generation, embeddings, vector stores |
| [Stage 2](stage-2-agentic-ai/index.md) | Agentic AI | Tools, planning, multi-agent systems |
| [Stage 3](stage-3-evaluation/index.md) | Evaluation | Measuring quality, LLM-as-judge, benchmarks |
| [Stage 4](stage-4-llmops/index.md) | LLMOps & Production | Serving, monitoring, cost, reliability |
| [Stage 5](stage-5-fine-tuning/index.md) | Fine-tuning | LoRA, SFT, preference tuning |
| [Stage 6](stage-6-expert/index.md) | Expert | Research frontier and specialization |
